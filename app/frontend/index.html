PPO Trading System

Welcome to the PPO Trading System, a comprehensive platform for training and testing trading models using Proximal Policy Optimization (PPO) reinforcement learning.

Key Features

  • PPO Reinforcement Learning: Utilize state-of-the-art reinforcement learning algorithms to optimize trading strategies.
  • Market Return Optimization: Train models that aim to outperform market returns.
  • Ulcer Performance Index (UPI): Protect equity from drawdowns with UPI optimization.
  • GPU Acceleration: Leverage CUDA/GPU processing for faster training.
  • Comprehensive Backtesting: Evaluate model performance with detailed metrics and visualizations.
  • Strategy Comparison: Compare PPO RL strategies with traditional PPO indicator strategies.

Documentation

PPO Trading System

The PPO Trading System is a comprehensive platform for training and testing trading models using Proximal Policy Optimization (PPO) reinforcement learning.

System Components

  • Trading Environment: Custom Gymnasium environment with buy/sell/hold actions, technical indicators, and reward function based on excess returns over market benchmark.
  • PPO Agent: StableBaselines3 implementation with configurable hyperparameters, neural network architecture, and TensorBoard logging.
  • Web Interface: Parameter configuration form for training models, model management, and backtesting interface.
  • Docker Configuration: CUDA/GPU support using nvidia/cuda base image and Docker Compose setup for easy deployment.

Performance Metrics

Metric Description
Total Return Total percentage return over the entire period
Annual Return Annualized percentage return
Volatility Annualized standard deviation of returns
Sharpe Ratio Risk-adjusted return (annual return / volatility)
Sortino Ratio Risk-adjusted return using downside deviation
Max Drawdown Maximum percentage drop from peak to trough
Ulcer Index Measure of downside risk based on drawdowns
Ulcer Performance Index (UPI) Risk-adjusted return using Ulcer Index (excess return / Ulcer Index)

API Documentation

The PPO Trading System provides a RESTful API for training and testing models. The API endpoints are:

  • /train - Train a new model
  • /tasks/{task_id} - Get task status
  • /models - List all models
  • /models/{model_id} - Get model information
  • /backtest - Backtest a model
  • /plots/{model_id}/{plot_name} - Get plot for a model
  • /compare - Compare strategies
  • /plots/{plot_name} - Get comparison plot
  • /documentation - Get API documentation

Docker Deployment

The PPO Trading System can be deployed using Docker with CUDA/GPU support. To deploy the system:

docker-compose up -d

This will start the system with GPU acceleration if available.
